Getting Started - top

Ahh, So you ready or not here it comes.Adventure is a clean & simple template, written in valid HTML 5, and powered by lots of cool JQuery and CSS 3 Features, but you already know that.What you don't know is how to use this template to make the best out of it, and that's why there's this awesome help file to get you started on your very own "Adventure"....so let's keep going!

Installation

Adventure is HTML template so there's no really "installation" required just unzip the main file you've downloaded from themeforest and after unziping you have your all template ready to be used.Just enter the "Adventure-html" folder and there you have all the files.

Setting Up The Grid

Before starting to work deeper with Adventure, you'll need to know how to use template's grid, it's simple and easy to use.Adventure is being build using one of the most popular grid, the 960 grid, which means your site container is 960px wide.To make it work without problem you be using columns for everything starting from sliders , portfolio , blog and more.

Setting Up the Homepage

Adventure template comes with 9 homepage styles.Every style has it's own slider or different look.The default homepage named "index.html" comes with the simple Nivo slider.

CSS in the Template - top

Changing The Style

You get 20 styles to choose from, 40 backgrounds and 40 Custom Fonts.Here's how to change it.

The default style is "style.css", if you want to change it just replace it with the style You want to use.

	<link rel="stylesheet" href="css/style.css" type="text/css" media="screen"/>

If You want to use for example "pure-red-theme" style just rename it.

	<link rel="stylesheet" href="css/pure-red-theme.css" type="text/css" media="screen"/>

To change the main background of your site, edit the style you're using and change the body part

	body {
	background-color:#FFF;	
	background-image:url(../images/body-bg-3.png);
	background-repeat:repeat;
	font: 12px/20px  'Droid Sans', arial, serif;
	line-height:20px;
	color: #666;
	/* Prevents Mobile Safari from bumping up font sizes in landscape */
	-webkit-text-size-adjust: 100%; 
	-webkit-tap-highlight-color: rgb(255,255,0);
}

If You want to change the body font, replace the font part in the Body.


	font: 12px/20px  'Droid Sans', arial, serif;

If You want to change the font of the headings edit the css file youre using an change this part:

	h1,h2,h3,h4,h5,h6 {
	font-family:'Crimson Text',arial,serif;
	font-weight:normal;
	font-variant:normal;
	white-space:normal;
	color:#272727;
	margin-bottom:10px;
}
h1 a:hover,h2 a:hover,h3 a:hover,h4 a:hover,h5 a:hover,h6 a:hover {
	color:#d83737;
	font-family:'Crimson Text',arial,serif;
	font-size:100%;
	text-decoration: none;
}

Twitter Feed

So in the begining of this help file i've showed you how to insert a HTML code for twitter feed to work.Now You need a little bit of Jquery to make it wokr just fine .What You'll have to do is to change the username of the twitter feeds, and if you want, the number of twitter feeds that will be displayed.

	/***************************************************
		TWITTER FEEDS
	***************************************************/
	jQuery.noConflict()(function($){
	$(document).ready(function() { 
	$(".tweet").tweet({
		username: "trendywebstar",
		join_text: null,
		avatar_size: null,
		count: 1,
		auto_join_text_default: "we said,", 
		auto_join_text_ed: "we",
		auto_join_text_ing: "we were",
		auto_join_text_reply: "we replied to",
		auto_join_text_url: "we were checking out",
		loading_text: "loading tweets..."
		});
		});
	});

You'll have to replace username "trendywebstar" with your own, and you can change the "count:3" wich represents the number of tweets.

		jQuery(document).ready(function($){
		var 
		speed = 700,   // animation speed
		$wall = $('#portfolio').find('.portfolio-grid ul')
		;

		$wall.masonry({
		singleMode: true,

		// only apply masonry layout to visible elements, if you want to apply this to one-third columns, change ".one-fourth:not" to ".one-third:not" below
		itemSelector: '.one-fourth:not(.invis)',
		animate: true,
		animationOptions: {
		duration: speed,
		queue: false
		}
		});

		$('#filterable a').click(function(){
		var colorClass = '.' + $(this).attr('class');

		if(colorClass=='.all') {
		// show all hidden boxes
		$wall.children('.invis')
		.toggleClass('invis').fadeIn(speed);
		} else {  
		// hide visible boxes 
		$wall.children().not(colorClass).not('.invis')
		.toggleClass('invis').fadeOut(speed);
		// show hidden boxes
		$wall.children(colorClass+'.invis')
		.toggleClass('invis').fadeIn(speed);
		}
		$wall.masonry();

		 return false;
		});

		});
